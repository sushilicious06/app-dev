# **Bridgerton** {#bridgerton-showcase}

## 🌸 Overview
*Bridgerton* is a **period drama series** created by Chris Van Dusen and produced by Shondaland.  
It premiered on **Netflix in 2020** and is based on Julia Quinn’s novels.  
Here’s a sentence with a footnote.[^1]

[^1]: Bridgerton is inspired by Julia Quinn’s book series, blending Regency-era romance with modern storytelling.
